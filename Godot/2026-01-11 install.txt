Install Godot
  Download Godot with C# support from https://godotengine.org/download/linux/ .
    https://downloads.godotengine.org/?version=4.5.1&flavor=stable&slug=mono_linux_x86_64.zip&platform=linux.64
  Extract into `/home/val/.local/bin/`.
  Create `/home/val/.local/share/applications/Godot.desktop` with the following content, adjusting values of Name and Exec.
```
#!/usr/bin/env xdg-open

[Desktop Entry]
Type=Application
Name=Godot 4.5.1
Exec="/home/val/.local/bin/Godot_v4.5.1-stable_mono_linux_x86_64/Godot_v4.5.1-stable_mono_linux.x86_64" %U
Categories=Application;
Name[en_US]=Godot
```
